Notițe din Programarea în Limbajul C/C++ pentru liceu, vol. I
===============================================================================

- algoritm = o succesisiune de pași prin care prelucrezi niște date (de
  intrare) și obții niște rezultate (date de ieșire)

  Exemple: REȚETA de a face o plăcintă - pas. 1, pas. 2...
              - pașii și ordinea lor
              - ingredientele - datele de intrare
              - produsul obținut (plăcinta) - date de ieșire

           SUMA a două numere
              - se dau: A și B (date de intrare)
              - rezultă: C (date de ieșire)
              - pașii: C = A + B

- limbaj de programare = mijloc de comunicare între programator și calculator
- program = descrierea unui algoritm într-un limbaj de programare
- (1) cod-mașină = limbajul procesorul
- (2) limbaj de asamblare
- (3) limbaj de programare de nivel înalt (nu ai nevoie de cunoștințe specifice
                                       de hardware)
- exemple de limbaje de progrmare: FORTRAN, Pascal, C, C++, C#, Java etc

Identificatori ( nume ) = rolul lor este de a denumi elemente din program: constante, variabile, functii, ... ; iar sintactic este o succesiune de cuvinte, litere,
                            caractere ( cum ar fi _ underscore ) iar primul carectere trebuie sa fie litera sau chiar underscore
                      - sunt luate in considerare primele 31 de caractere desi poate sa aiba orice lungime
                      - lfimbajul C/C++ este de tip case-sensitive - face diferenta dintre literele mici si mari
                      - pentru a intelege scopul folosirii acestora este bine sa se foloseasca identificatorii sugestivi si este bine sa se construiasca mai multe
                      radacini sau cuvinte iar in acest caz se pot folosi majuscule ( ex. NrDeLaMasina, NrTelefon, etc )
 
Cuvinte rezervate ( cuvinte cheie = keywords ) = au inteles predefinit si sunt utilizate in constructii sintactice in care sunt specificati
       Exempele =>  if - descrie inceputul unei instructiuni alternative
                    while - descrie inceputul unei instructiuni repetitive

Comentarii = sunt ignorate de catre compilaotr ( este un program cu rol de a traduce textul programului care este scris intr-un limbaj de programare numit cod sursa
in limbaj-masina iar astfel se obtine cod-obiect ) iar ca si rol are de a explica pentru progarmatori anumite secvente din program
    Exemple: - /**/ se poate scrie mai multe linii de cod
             - // se poate scrie doar o singura linie de cod
